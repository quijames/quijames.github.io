How to install an SSL certificate on Azure Websites

Posted on Aug 26, 2015
#IIS #SSL #Azure
  1. Buy a UCC SSL certificate on GoDaddy.
  2. To set it up, you’ll need to create a CSR. Do this on your local IIS. See this link on how to do this.
  3. Paste the CSR into GoDaddy, and enter the other domains in the Subject Alternate Names (SAN’s) field on GoDaddy
  4. Once the domains have been verified, download the certificate.
  5. Install the certificate on local IIS - follow this guide
  6. Once installed, open up IIS, and open “Server Certificates”. Right clik on the certificate, and click “Export”.
  7. Export it to a .pfx file and put a password on it.
  8. Manage your website in Azure, and go to the “Configure” tab.
  9. Upload the SSL certificate, and update the bindings, and off you go!
