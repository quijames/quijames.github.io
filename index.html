Posted on Jul 10, 2015
Part 1: Setting up your environment Before starting anything, you need to set up Android on your machine. Fixing [ERROR] initializing cordova class not found This is a result of running cordova plugin add org.apache.cordova.device No worries, just add the following to your /www/config.xml: <feature name="Device"> <param name="android-package" value="org.apache.cordova.device.Device" /> </feature> When Phonegap builds for Android now, it will add this to /platforms/android/res/xml/config.xml.
